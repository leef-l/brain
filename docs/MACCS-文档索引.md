## 文档更新规则

1. **新功能必须先写文档，后写代码**
2. **代码变更后必须同步更新文档**
3. **每个 Wave 完成后必须更新本文档索引**
4. **旧文档标记落后状态时，必须注明替代文档**
5. **实验数据、构建产物不得纳入 git**
