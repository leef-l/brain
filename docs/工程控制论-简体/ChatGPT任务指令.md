## 处理规范（严格遵守）

### 1. 角色定位

你在做"**修复与转写**"，不是"翻译"或"改写"。保留钱学森原文的论述风格、句式、术语，**只修 OCR 噪声和繁简**。

### 2. OCR 错字修正

OCR 文本典型错误：
- 「鈸 f 在著」实为「錢學森著」
- 「乃ft 尸」实为「乃氏圖」
- 「⑺ 時」实为「t → ∞ 時」
- 「f ⑹」实为「k(ξ)」

规则：
- **能根据上下文判断** → 静默改正
- **不能判断** → 保留原字形 + `[OCR?]`
- **整段损坏无法还原** → 保留可识别部分 + `[……OCR 破损……]`

### 3. 繁简转换

全文繁体转简体。专有名词见下方"术语对照表"，**优先保留原书 1958 年译法**。

### 4. 数学公式

OCR 对公式破坏极严重。处理方式：
- 能推断的 → LaTeX 还原：行内 `$...$`，独立 `$$...$$`
- 带编号的 → 保留编号 `\tag{3.41}` 或 `(3.41)`
- 推断不出的 → 标 `[公式 X.Y：OCR 破损]`
- **禁止编造公式**

### 5. 图表

- 图表在 OCR 中全部丢失
- 引用处保留 `**图 X.Y**` 粗体占位
- 不要描述或臆测图的内容

### 6. 参考文献与脚注

- 章末脚注整理成 Markdown 列表
- 俄文字符保留（如 А. М. Ляпунов）
- 英文文献格式：作者, *期刊*, 卷, 页码 (年份)
